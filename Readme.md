LemonStatus Creater (MEAN stack)
=========
**레몬스테이터스 생성기 ProtoType**

다수의 유닛을 기획할 경우,

간단하게 스탯을 생성하고, 사용자 로그인 후

MongoDB에 등록할 수 있도록 만든 툴입니다.

테스트 해 보실려면 [Heroku 앱 테스트][app] 을 눌러주세요.
